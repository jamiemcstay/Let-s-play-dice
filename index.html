Let's Play Dice

Game Rules

Each player will roll one die to start to start the game.
The player who rolls the highest number is the first banker in the game play.

Betting

The banker gets to set the stakes before the round, each player has a bankroll of $500.

Objective

The objective of the game is to roll the better dice combination.

Dice Combinations

  • Instant Win: 4-5-6; three-of-a-kind; Pair + 6
  • Instant Loss: 1-2-3; Pair + 1
  • Set point: Pair + 2; Pair + 3; Pair + 4; Pair + 5

Game Play

If the banker rolls an instant win combination, they win all the bets made.

If the banker rolls an instant loss combination, they loses the bets made.

If the banker sets a point, the other player will get a turn to roll the dice.

If a player gets an instant win or a better point, they win the round.

Rules

Gameplay continues until either players bankrolls reaches $0

When a player wins a round they are automatically banker in the next round.
